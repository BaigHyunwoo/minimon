

본 프로젝트는 서비스에 도입하기 전 테스트 용도로 만든 연습용 프로젝트입니다.

지정된 시간마다 저장해놓은 사이트를 접근하거나 API 호출을 하여 정상적으로 동작하는지 검사하는 시스템입니다.

chromedriver와 chrome browser, selenium을 이용하여 만든 모니터링 시스템입니다.

프로젝트를 실행하기 위해선 chromeDriver를 설치한 후 해당 파일 위치를 저장해야 합니다.

chromedriver는 https://chromedriver.chromium.org/downloads 에서 자신의 PC에 설치된 chrome browser와 일치하는 버젼으로 다운로드 받아야 합니다.
