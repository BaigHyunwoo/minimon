

본 프로젝트는 실 서비스에 도입하기 전 테스트 용도로 만든 연습용 프로젝트입니다.

지정된 시간마다 저장해놓은 사이트에 접근하거나 Http 호출을 하여 정상적으로 동작하는지 검사하는 시스템입니다.

chromedriver와 chrome browser, selenium을 이용하여 만든 모니터링 시스템입니다.

프로젝트를 실행하기 위해선 selenium plugin과 chromeDriver를 설치가 필요합니다.

chromedriver는 https://chromedriver.chromium.org/downloads 에서 자신의 PC에 설치된 chrome browser와 일치하는 버젼으로 다운로드 받아야 합니다.

chromeDriver를 설치 한 후엔 application.yml파일에 해당 폴더 경로를 넣어주세요.
